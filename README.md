# Done. — Task Manager

Small single-page ToDo app used for the Sytent Technologies internship task.

## What this folder contains

- `todo_app.html` — Primary HTML file (interactive ToDo app).
- `style.css` — External stylesheet (styles were extracted from the HTML).

## Usage

1. Open `todo_app.html` in a browser (double-click or serve from a static host).
2. Add tasks, mark done, filter, and clear completed items. Tasks persist in `localStorage`.
